# Japanese_Subtitle_Tool 🎙️

这是一个专为日语播客/音频设计的转文字工具，配备了实用的网页控制台、丝滑的进度条显示，并支持 4060 等 NVIDIA 显卡的 CUDA 硬件加速！

## ✨ 核心功能
1. **音频一键转文字**：支持将 `.mp3`, `.m4a`, `.wav` 等格式一键转换为带时间戳的 `.txt` 文本或标准播放器 `.srt` 字幕。
2. **4060 显卡全开**：完美适配本地 CUDA 加速，加载 Whisper medium 模型全速运行。
3. **附加利器（已有 TXT 秒转 SRT）**：如果你之前转过了 TXT，不需要重新跑音频，直接上传 TXT 文本，0.1 秒智能离线提取生成 SRT 字幕。

## 🛠️ 本地运行指南
1. 克隆或下载本仓库代码。
2. 建议使用 Python 12 绿色版环境或虚拟环境。
3. 安装依赖（确保配置好 PyTorch 的 CUDA 版本）。
4. 运行 `python audio_to_text.py` 即可自动拉起本地浏览器控制台。
